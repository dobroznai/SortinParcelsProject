#### clean очистка артефактов

#### validate проверка структуры, все ли корректно

#### compile компиляция java кода в .class

#### test запускает unit тесты

#### package упаковка в .jar

#### verify проверка все ли тесты выполнились успешно

#### install учтановка приложения (артефакта) в локальный репозиторий

#### site генерация документации

#### deploy отправка готового артефакта на удаленный репозиторий

### Commands
#### mvn dependency:tree просмотр дерево зависимостей
#### mvn dependency:purge-local-repository -очищает и переустанавливает то что находиться в локальном репозитории
#### mvn help:effective-pom - смотрит конфликт библиотек и какие библиотеки используються
####  mvn spring-boot:run - запуск строки
#### mvn clean verify -DskinTests --> если команда не готова может згенерировать без тестов
#### mvn test --> surefire:report посмотреть
